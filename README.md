# Fitness Center FAQ Bot

A React-based web application for a fitness center featuring an AI-powered chatbot assistant.

## Features

- **Interactive Chatbot**: Powered by Google Gemini AI to answer questions about facilities, programs, and memberships.
- **Gym Information**: Detailed sections about expert trainers, premium amenities, and operating hours.
- **Contact Form**: Easy-to-use contact form for inquiries.
- **Responsive Design**: Modern, dark-themed UI with "Red-Black" aesthetics.

## Tech Stack

- **Frontend**: React, Vite
- **Styling**: Vanilla CSS (Custom properties, Flexbox/Grid)
- **AI Integration**: Google Gemini API
- **Routing**: React Router DOM

## Setup

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file and add your Gemini API key:
   ```
   VITE_GEMINI_API_KEY=your_api_key_here
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
